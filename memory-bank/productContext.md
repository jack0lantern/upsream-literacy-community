# Product context

## Problem

District literacy leaders are isolated. Generic forums and conferences rarely match **district context** (size, locale, poverty, ELL) with **specific problems** (e.g., SoR transition, intervention staffing). Upstream targets **structured, context-aware matching**.

## Primary users

- Literacy directors, CAOs, curriculum coordinators, literacy coaches, MTSS/RTI leads with literacy responsibility.

## Key flows (MVP)

1. **Onboarding** — Sign up → state + district (with auto-filled stats where possible) → role → 1–5 problem statements → optional bio → dashboard with initial matches.
2. **Discovery** — Match cards (peer + district stats + shared problems + score), filters and sorts, profile view.
3. **Messaging** — Start thread from match/profile, async conversation, mute/report.
4. **Moderation** — Admin queue for flags, thread view, warn/suspend/delete/close.

## Personas (summary)

- **Maria** — Suburban literacy director; needs SoR transition peers in similar districts.
- **James** — Rural coordinator; high FRL, small district; intervention staffing on a budget.
- **Denise** — Upstream staff moderator; needs visibility and tooling without reading everything.
