# Project brief

## Name

Upstream Literacy Community — peer-matching for district literacy leaders.

## Purpose

Connect district literacy leaders with peers in **demographically similar** districts who share **the same operational challenges**, using NCES-style public district data plus self-selected problem statements. Enable structured discovery and 1:1 messaging (MVP scope per PRD).

## Goals (MVP)

- Onboarding: account, district selection (autocomplete / NCES-backed), role, problem statements, optional bio.
- Matching dashboard: ranked matches, filters, profile drill-down.
- Messaging: 1:1 threads, notifications (email + in-app per PRD).
- Admin moderation: flagged content, actions on users/messages.

## Non-goals (unless PRD changes)

- Not a generic social network or vendor-led community.
- Scope boundaries are defined in [docs/PRD.md](../docs/PRD.md).

## Success criteria (from product intent)

Leaders find context-relevant peers quickly; Upstream builds an engaged network with paths to subscription/PD/marketplace (business context in PRD).
